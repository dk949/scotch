## Build Dependencies

### Required

* D compiler
  * Tested on DMD 2.100.0
* DUB
  * Tested with version 1.29
* make
  * Only required on POSIX systems

## Runtime dependencies

A JavaScript runtime is needed to execute generates code.

## Building

Build the project either with DUB

``` sh
dub build
```

or with make

``` sh
make build
```
